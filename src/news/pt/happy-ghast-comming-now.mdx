### Principais pontos
- O Happy Ghast está disponível apenas na versão de visualização do Minecraft Bedrock; a versão snapshot do Java Edition ainda não inclui este recurso.
- A versão de visualização Bedrock 1.21.80.25 (lançada em 3 de abril de 2025) já inclui o Happy Ghast.
- Os usuários podem experimentar o Happy Ghast na versão de visualização Bedrock, mas os jogadores do Java Edition podem precisar esperar por atualizações futuras.

### Como baixar e instalar a versão de visualização Bedrock
Para experimentar a versão de visualização do Minecraft que inclui o Happy Ghast, siga estas etapas:
- **Baixe e instale o Minecraft Launcher**: Visite o site oficial [Minecraft](https://www.minecraft.net/en-us/get-minecraft) para baixar o launcher para sua plataforma (Windows, macOS ou Linux) e siga as instruções de instalação.
- **Ative a versão de visualização Bedrock**: Abra o Minecraft Launcher, vá para a aba "Minecraft Bedrock", selecione a versão "Latest Preview" e clique em "Install" para instalar ou "Play" para iniciar.
- **Crie ou carregue um mundo**: Na versão de visualização Bedrock, crie um novo mundo ou carregue um existente, garantindo que as atualizações mais recentes e os recursos experimentais necessários estejam ativados (se solicitado).

### Como obter e usar o Happy Ghast no jogo
Após entrar na versão de visualização, siga estas etapas para experimentar o Happy Ghast:
- **Obtenha o Bloco de Ghast Seco**: Vá para o Nether e explore os vales de areia das almas para encontrar Blocos de Ghast Seco, ou crie-os com 5 blocos de osso e 4 lágrimas de ghast.
- **Transforme o Bloco de Ghast Seco em um Ghast Jovem**: Leve o Bloco de Ghast Seco para o Overworld, coloque-o no chão e molhe-o com água, aguardando cerca de 20 minutos para que ele se transforme em um Ghast Jovem.
- **Faça o Ghast Jovem crescer em um Happy Ghast**: Alimente o Ghast Jovem com bolas de neve para acelerar seu crescimento. Quando crescer, ele se tornará um Happy Ghast.
- **Crie e equipe uma Sela de Ghast**: Use 3 couros, 2 blocos de vidro e 1 bloco de lã para criar uma Sela de Ghast (você pode personalizar a cor com corantes). Clique com o botão direito no Happy Ghast para equipar a sela.
- **Monte no Happy Ghast**: Clique com o botão direito no Happy Ghast equipado com a sela para montá-lo. Ele pode carregar até 4 jogadores e é controlado com bolas de neve.

Lembre-se de que esta é uma versão de teste e pode conter bugs ou instabilidades. Recomenda-se fazer backup do mundo antes de experimentar.

O Happy Ghast faz parte da atualização de 2025 do Minecraft, anunciada inicialmente no Minecraft Live 2025 e prevista para lançamento oficial no segundo semestre de 2025. No entanto, a versão snapshot do Java Edition (como a 25w10a, lançada em 5 de março de 2025) ainda não inclui este recurso. Os jogadores do Java Edition precisarão esperar por atualizações futuras.

#### Etapas para baixar e instalar a versão de visualização Bedrock
Para experimentar o Happy Ghast na versão de visualização, certifique-se de ter o Minecraft Launcher instalado e ative a versão de visualização Bedrock. As etapas são:
- **Baixar e instalar**: Visite [Minecraft](https://www.minecraft.net/en-us/get-minecraft) para baixar o launcher para Windows, macOS ou Linux e siga as instruções de instalação.
- **Ativar a versão de visualização**: Abra o launcher, vá para a aba "Minecraft Bedrock", selecione "Latest Preview" e clique em "Install" ou "Play".
- **Criar ou carregar um mundo**: No jogo, crie um novo mundo ou carregue um existente. Recomenda-se fazer backup do mundo para evitar perda de dados devido a possíveis bugs.

#### Processo detalhado para obter e usar o Happy Ghast no jogo
O Happy Ghast pode ser obtido seguindo estas etapas:
1. **Obter o Bloco de Ghast Seco**:
    - Explore os vales de areia das almas no Nether para encontrar Blocos de Ghast Seco.
    - Ou crie-o com 5 blocos de osso e 4 lágrimas de ghast em uma bancada de trabalho.

2. **Transformar em Ghast Jovem**:
    - Leve o Bloco de Ghast Seco para o Overworld e coloque-o no chão.
    - Use um balde de água para molhá-lo e aguarde cerca de 20 minutos para que ele se transforme em um Ghast Jovem.

3. **Crescer para Happy Ghast**:
    - Alimente o Ghast Jovem com bolas de neve para acelerar seu crescimento.
    - Sem alimentação, ele ainda crescerá, mas levará mais tempo.

4. **Criar e equipar a Sela de Ghast**:
    - Use 3 couros, 2 blocos de vidro e 1 bloco de lã para criar a Sela de Ghast.
    - Corantes podem ser usados para personalizar a cor da sela.
    - Clique com o botão direito no Happy Ghast para equipar a sela.

5. **Montar e controlar**:
    - Clique com o botão direito no Happy Ghast equipado para montá-lo.
    - Até 4 jogadores podem montar, e o movimento é controlado com bolas de neve.

Verifique se seu dispositivo é compatível.
