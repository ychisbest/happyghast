### Points clés
- Le Ghast heureux (Happy Ghast) est actuellement disponible uniquement dans la version préliminaire de Minecraft Bedrock. La version snapshot Java ne comprend pas encore cette fonctionnalité.
- La version préliminaire Bedrock 1.21.80.25 (publiée le 3 avril 2025) inclut désormais le Ghast heureux.
- Les utilisateurs peuvent expérimenter le Ghast heureux via la version préliminaire Bedrock, mais les joueurs Java devront probablement attendre des mises à jour ultérieures.

### Comment télécharger et installer la version préliminaire Bedrock
Pour essayer la version préliminaire de Minecraft incluant le Ghast heureux, suivez ces étapes :
- **Téléchargez et installez le lanceur Minecraft** : Rendez-vous sur le site officiel [Minecraft](https://www.minecraft.net/en-us/get-minecraft) pour télécharger le lanceur adapté à votre plateforme (Windows, macOS ou Linux), puis suivez les instructions d'installation.
- **Activez la version préliminaire Bedrock** : Ouvrez le lanceur Minecraft, allez dans l'onglet "Minecraft Bedrock", sélectionnez la version "Latest Preview", puis cliquez sur "Install" ou "Play" pour lancer le jeu.
- **Créez ou chargez un monde** : Dans la version préliminaire, créez un nouveau monde ou chargez un monde existant, en veillant à activer les dernières mises à jour et les fonctionnalités expérimentales si nécessaire.

### Comment obtenir et utiliser le Ghast heureux en jeu
Une fois dans la version préliminaire, voici comment profiter du Ghast heureux :
- **Obtenez un bloc de Ghast séché** : Rendez-vous dans le Nether et explorez les vallées des sables des âmes pour trouver des blocs de Ghast séchés. Vous pouvez aussi en fabriquer avec 5 blocs d'os et 4 larmes de Ghast.
- **Transformez le bloc de Ghast séché en bébé Ghast** : Ramenez le bloc dans le monde normal, placez-le au sol et arrosez-le d'eau. Attendez environ 20 minutes pour qu'il se transforme en bébé Ghast.
- **Faites grandir le bébé Ghast en Ghast heureux** : Nourrissez le bébé Ghast avec des boules de neige pour accélérer sa croissance. Une fois adulte, il deviendra un Ghast heureux.
- **Fabriquez et équipez un harnais pour Ghast** : Utilisez 3 cuirs, 2 blocs de verre et 1 bloc de laine pour fabriquer un harnais (vous pouvez le teindre). Faites un clic droit sur le Ghast heureux pour lui équiper le harnais.
- **Montez le Ghast heureux** : Faites un clic droit sur le Ghast équipé pour le monter. Jusqu'à 4 joueurs peuvent s'y installer, et les boules de neige permettent de le diriger.

Notez qu'il s'agit d'une version de test, susceptible de contenir des bugs. Il est recommandé de sauvegarder vos mondes avant de l'essayer.

### Rapport détaillé
Ce guide complet explique comment expérimenter le Ghast heureux dans la version préliminaire de Minecraft, en couvrant les informations contextuelles, les différences entre versions, les étapes d'obtention et les précautions à prendre.

#### Contexte et différences entre versions
Le Ghast heureux fait partie de la mise à jour 2025 de Minecraft, annoncée lors de Minecraft Live 2025. Sa sortie officielle est prévue pour l'été 2025. À la date du 4 avril 2025, le Ghast heureux est disponible dans la version préliminaire Bedrock 1.21.80.25, publiée le 3 avril 2025 [Minecraft Bedrock Edition Preview 1.21.80.25 Update Notes](https://www.minecraft.net/en-us/article/minecraft-bedrock-1-21-80-25). En revanche, la version snapshot Java (comme la 25w10a, sortie le 5 mars 2025) ne l'inclut pas encore.

#### Étapes pour télécharger et installer la version préliminaire Bedrock
Pour accéder au Ghast heureux, assurez-vous d'avoir le lanceur Minecraft et activez la version préliminaire Bedrock :
- **Téléchargement et installation** : Allez sur [Minecraft](https://www.minecraft.net/en-us/get-minecraft) pour télécharger le lanceur adapté à Windows, macOS ou Linux, puis installez-le.
- **Activation de la version préliminaire** : Dans le lanceur, allez dans l'onglet "Minecraft Bedrock", choisissez "Latest Preview", puis cliquez sur "Install" ou "Play".
- **Création ou chargement d'un monde** : Dans le jeu, créez un nouveau monde ou chargez-en un existant. Pensez à sauvegarder vos mondes au préalable.

#### Processus détaillé pour obtenir et utiliser le Ghast heureux
Voici comment obtenir le Ghast heureux étape par étape :
1. **Obtenir un bloc de Ghast séché** :
    - Explorez les vallées des sables des âmes dans le Nether pour en trouver.
    - Sinon, fabriquez-en avec 5 blocs d'os et 4 larmes de Ghast dans un établi.

2. **Transformation en bébé Ghast** :
    - Ramenez le bloc dans le monde normal et placez-le au sol.
    - Utilisez un seau d'eau pour l'arroser. Après environ 20 minutes, il deviendra un bébé Ghast.

3. **Croissance en Ghast heureux** :
    - Nourrissez le bébé Ghast avec des boules de neige pour accélérer sa croissance.
    - Sans intervention, il grandira naturellement en Ghast heureux, une monture volante amicale pouvant transporter jusqu'à 4 joueurs.

4. **Fabrication et équipement du harnais** :
    - Dans un établi, combinez 3 cuirs, 2 blocs de verre et 1 bloc de laine pour fabriquer un harnais.
    - Utilisez des teintures pour personnaliser sa couleur.
    - Faites un clic droit sur le Ghast heureux pour lui équiper le harnais.

5. **Monte et contrôle** :
    - Faites un clic droit sur le Ghast équipé pour le monter.
    - Utilisez des boules de neige pour le diriger et profitez du vol.

#### Précautions et problèmes potentiels
- **Stabilité de la version** : La version préliminaire peut contenir des bugs ou endommager les mondes. Sauvegardez-les avant de l'utiliser.
- **Compatibilité des appareils** : Selon [Minecraft Preview 1.21.80.25 | Minecraft](https://www.minecraft.net/en-us/article/minecraft-preview-1-21-80-24), les appareils supportés incluent Xbox Series X|S, Xbox One, PlayStation 4, PlayStation 5, Android, iOS et PC. Vérifiez que votre appareil est compatible.
- **Joueurs Java** : Le Ghast heureux n'est pas encore disponible en Java. Attendez les prochaines snapshots.
